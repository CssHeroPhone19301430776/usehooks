## ✨ 特性

- 易学易用
- 包含丰富的基础 Hooks
- 可靠的代码健壮：使用 Typescript 构建，提供完善的类型定义文件
- 完善的文档能力：支持文档记录，支持 demo 演示

## 📦 安装

:::code-group

```bash [npm]
npm install @csshero/usehooks
```

```bash [yarn]
yarn add @csshero/usehooks
```

```bash [pnpm]
pnpm add @csshero/usehooks
```

:::

## 🔨 使用

```ts
import { useToggle } from '@csshero/usehooks';
```
